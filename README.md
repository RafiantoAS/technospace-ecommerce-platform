# TechnoSpace Ecommerce Platform

Modern cyberpunk-inspired ecommerce platform for gaming peripherals developed with HTML, CSS, JavaScript, and jQuery.

## Overview

TechnoSpace is a multi-page frontend ecommerce website designed to showcase premium gaming peripherals through a futuristic cyberpunk user interface. The project focuses on responsive web design, interactive navigation, product presentation, image galleries, and client-side form validation.

The application simulates a modern online gaming store where users can explore gaming keyboards, gaming mice, product galleries, and register for a TechnoSpace account through an intuitive and visually engaging experience.

This project was developed to strengthen frontend development skills while applying modern UI design principles and responsive layouts.

## Pages

### Home Page

The landing page introduces the TechnoSpace brand, featured products, statistics, promotional content, and an interactive image slider.

### Gallery Page

Displays gaming setups, peripherals, RGB battlestations, and cyberpunk-inspired product showcases.

### Keyboard Collection

Showcases premium mechanical keyboards with product information, switch types, pricing, and product highlights.

### Mouse Collection

Features professional gaming mice with specifications, product descriptions, and pricing information.

### Register Page

Allows users to register through a validated form containing personal information and product preferences.

## Challenges and Solutions

### Responsive Layout Design

Challenge:
Maintaining a consistent user experience across multiple screen sizes.

Solution:
Implemented responsive layouts using Flexbox, CSS Grid, and media queries.

### Interactive Slider

Challenge:
Creating smooth image transitions without disrupting page performance.

Solution:
Developed a JavaScript-based slider with controlled transitions and automatic slide rotation.

### Navigation Experience

Challenge:
Ensuring dropdown menus remain accessible and user-friendly.

Solution:
Improved hover behavior and navigation spacing for a smoother browsing experience.
